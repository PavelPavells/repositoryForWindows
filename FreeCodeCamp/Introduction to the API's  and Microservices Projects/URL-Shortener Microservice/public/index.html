How to create a new shortened URL:

Go to

https://shortthis.herokuapp.com/new/

And pass a URL in the following format

http(s)://www.example.com

For Example:

https://shortthis.herokuapp.com/new/https://www.google.com

Output:

{
  originalURL: https://www.google.com/,
  shorthandURL: https://shortthis.herokuapp.com/UfT
}

How to use the created shortened URL:

Just go to the created shortened URL:

https://shortthis.herokuapp.com/UfT

This will automatically redirect you to the former original URL
