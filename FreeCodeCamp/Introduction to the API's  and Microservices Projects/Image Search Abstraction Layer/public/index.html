How to search for images

Go to

https://morning-basin-53123.herokuapp.com/api/imagesearch/

and pass a search term and an optional offset-value (offset defaults to 1)

<searchTerm> [?offset= <nr> ]

Examples:

.../api/imagesearch/freeCodeCamp .../api/imagesearch/freeCodeCamp?offset=10

Output:

An array containing 10 objects. Each object represents an image.

If you use the offset-parameter with a number n. The first image in the array will be the nth search result.

[...
 {
  snippet: "freeCodeCamp",
  imageURL: "https://avatars0.githubusercontent.com/...",
  siteURL: "https://github.com/FreeCodeCamp",
  thumbnail: "https://encrypted-tbn0.gstatic.com/images?..."
 }
...]

How to get the most recent searches

Just go to:

https://morning-basin-53123.herokuapp.com/api/latest/imagesearch

Output:

An array containing 10 objects. Each object represents a search.

[...
 {
  term: "freecodecamp",
  date: "Sun, 13 Jan 2019 15:43:28 GMT",
  timestamp: 1485777568
